# Docs

## Purpose
Operational and design documentation for the IaC framework Milestone 1 delivery.

## Contents
- `architecture.md`: component boundaries and flow.
- `gcloud-commands.md`: login and identity discovery commands.
- `iac-state/`: lightweight record of currently deployed infrastructure state.
- `input.md`: quick input reference.
- `inputs.md`: required and optional inputs.
- `makefile.md`: local Makefile command interface and safeguards.
- `validation.md`: deterministic validation sequence and advisory pattern-review boundary.
- `manual-commands.md`: human-run apply and destroy commands.
- `troubleshooting.md`: common issues and quick diagnostics.
